Arcane Studio

Smart productivity tools and automations for development teams

TL;DR

This application is a collection of productivity-boosting tools built on top of the Arcane Engine. The engine takes natural language instructions (Tasks) and executes them on your behalf using its capabilities, which include:

  • Reading, writing and editing code
  • Interacting with issues, tickets, wikis, etc via Integrations
  • Searching the internet and scraping websites

Get Started

The entire suite is free and open-source, and we welcome contributions from the community. If you're interested in building a tool like this for your own team, check out our Development Kit and CLI.

To give you an idea of what the engine can do for you, here are some examples from our Lab:

  • Read Helm chart and build scripts to generate docker-compose file
  • Break down a large module into smaller ones in seconds
  • Scrape project to generate beautiful README file
